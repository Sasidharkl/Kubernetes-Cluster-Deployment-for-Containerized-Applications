# Kubernetes-Cluster-Deployment-for-Containerized-Applications

## Project Overview

This project demonstrates deploying a production-ready Kubernetes cluster to manage containerized applications. The application is containerized using Docker and deployed to Kubernetes with automated CI/CD pipelines.

The deployment includes scalable pods, rolling updates, load balancing, and ingress configuration for high availability.

## Key Features

- Docker containerization
- Kubernetes deployment with pods and services
- Ingress-based traffic routing
- CI/CD pipeline using GitHub Actions
- Rolling updates and scalability
- Production-ready deployment structure

## Tech Stack

- Kubernetes
- Docker
- Python Flask
- GitHub Actions
- Nginx Ingress Controller

## Architecture

Client → Ingress → Kubernetes Service → Pods
